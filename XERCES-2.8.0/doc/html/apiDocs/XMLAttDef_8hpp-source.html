00448     // -----------------------------------------------------------------------
00449     //  Private data members
00450     //
00451     //  fDefaultType
00452     //      Indicates what, if any, default stuff this attribute has.
00453     //
00454     //  fEnumeration
00455     //      If its an enumeration, this is the list of values as space
00456     //      separated values.
00457     //
00458     //  fId
00459     //      This is the unique id of this attribute, given to it when its put
00460     //      into the validator's attribute decl pool. It defaults to the
00461     //      special value XMLAttrDef::fgInvalidAttrId.
00462     //
00463     //  fProvided
00464     //      This field is really for use by the scanner. It is used to track
00465     //      which of the attributes of an element were provided. Any marked
00466     //      as not provided (after scanning the start tag) and having a
00467     //      default type of Required, is in error.
00468     //
00469     //  fType
00470     //      The type of attribute, which is one of the AttTypes values.
00471     //
00472     //  fValue
00473     //      This is the value of the attribute, which is the default value
00474     //      given in the attribute declaration.
00475     //
00476     //  fCreateReason
00477     //      This flag tells us how this attribute got created.  Sometimes even
00478     //      the attribute was not declared for the element, we want to fault
00479     //      fault it into the pool to avoid lots of redundant errors.
00480     //
00481     //  fExternalAttribute
00482     //      This flag indicates whether or not the attribute was declared externally.
